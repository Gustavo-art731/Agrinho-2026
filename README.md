# Agro Forte, Futuro Sustentável 🌱

Projeto desenvolvido para o **Concurso Agrinho 2026** — Categoria Programação, Subcategoria 3 (Front-End).

## Tema

> "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"

## Objetivo

O site apresenta informações sobre sustentabilidade no agronegócio brasileiro, mostrando como é possível produzir alimentos de forma eficiente sem destruir o meio ambiente. O projeto aborda temas como agricultura de precisão, gestão da água, preservação de matas e economia circular.

## Funcionalidades

- **Página informativa** com os pilares da sustentabilidade no campo
- **Quiz interativo** com 5 perguntas sobre agro sustentável e pontuação ao final
- **Calculadora de Reserva Legal** baseada no Código Florestal Brasileiro (Lei nº 12.651/2012)
- **Contadores animados** com dados do agronegócio brasileiro
- **Design responsivo** adaptado para celulares e tablets

## Tecnologias Utilizadas

- HTML5 (estrutura semântica)
- CSS3 (flexbox, variáveis CSS, media queries, animações)
- JavaScript puro (manipulação do DOM, eventos, IntersectionObserver)
